Pour tester le programme

-- Prerequis 
Le client étant conforme à la norme pop, il peut comuniquer avec n'importe quels serveurs mails.
Pour ma part j'ai utilisé "Courier" sous linux (installé en local sur nom ordinateur).

Dans le dossier mail-type, j'ai fournis des mails types pour tester le logiciel. Il faut les ajouter a votre dossier de reception de votre serveur (local ou non).

Par l'utilisation d'un vrai serveur mail, je ne peux pas vous fournir de vrai procedure de teste mais il faut savoir que toute les fonctionnalitées ont étés implentés. 

-- compilation 
Pour pouvoir compiler le programme:
	make 
Pour compiler le programme et lancer une version du programme:
	make [t-test, c-test, g-test]
